  • favorite extensions
    • online safety and security
      • Privacy Badger - stops advertisers and other third-party trackers
      • Ghostery - block ads, stop trackers, speed up websites
      • Adblocker Ultimate / Adblock Plus - block ads
      • uBlock Origin - block ads
      • HTTPS Everywhere - automatically switches sites from insecure "http" to secure "https"
      • Location Guard - reports a fake location to websites, obtained by adding a certain amount of "noise" to the real location
    • for developers
      • Web Developer - adds a toolbar button to the browser with various web developer tools
      • Web Developer Checklist - analyses any web page for violations of best practices
      • CSS Peeper - simple CSS properties viewer by hovering over an element
      • HTML5 Outliner - provides an outline of the current page based on HTML5 sectioning elements and headings
      • Colorpick Eyedropper - zoomed eyedropper & color chooser tool that allows you to select color values from webpages
      • SEO Analysis & Website Review - quickly check any business’ SEO score without leaving the website
    • online shopping
      • Honey - automatically find and apply coupon codes
      • Rakuten - get Cash Back for shopping at over 2,500 stores
      • Fakespot - spots, analyzes and identifies fake reviews and counterfeits
